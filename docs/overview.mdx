Lusift is a javascript library for creating in-app guided tours for your web apps.

<b>Bundle size</b>: ~61kb

## Installation
Installation is just a simple command.
```bash
npm install lusift
```

### Requirements
`tslib` version `^1.9.0 || >=2.0.0`

`@floating-ui/dom` version `>=0.0.1`
