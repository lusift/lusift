This is the default css, you can use these selectors to globally customise the styling of your guide elements.

```css

    --primary-color: #17a3a8;
    --modal-border-radius: 4px;
    --tooltip-border-radius: 4px;
    --progress-bar-height: 3px;
}

    background-color: var(--primary-color);

    outline: 2px solid transparent;
    color: #fff;
    padding: 0.2rem 0.35rem;
    font-size: 0.75rem;
    font-weight: bold;
    border-radius: 8px;
}

.lusift .tooltip button:focus {
    outline-color: #444;
}

.lusift .modal button {
    font-size: 1.1rem;
    padding: 0.45rem 0.75rem;
}

    background-color: transparent;
    border: none;
    font-weight: normal;
    position: absolute;
    top: 1px;
    right: 0px;
}

    display: flex;
    justify-content: flex-end;
    margin: 0;
```
