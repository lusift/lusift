Creating Hotspot element

```
type Hotspot = {
    index: number;
    type: 'hotspot';
    target: Target;
    beacon: {
        placement: {
            top: number;
            left: number;
        };
        size: number;
        color: string;
        type: string;
    };
    tip: {
        data: {
            placement: Position;
            arrow: boolean;
            bodyContent: BodyContent;
        };
        styleProps: object;
    };
    async: boolean;
}
```
