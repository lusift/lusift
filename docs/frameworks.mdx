title: 'Frameworks'
description: Frameworks supported by Lusift
---

In addition to vanilla JS, Lusift supports <b>React</b> and <b>Vue</b>(version >2.6.11)

To use with React components, the import changes to:

```typescript
import Lusift from 'lusift/react';
```

For Vue:
```typescript
import Lusift from "lusiftx/dist/vue/src";
```

Rest stays the same.
